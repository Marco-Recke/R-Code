glmCI {wrappedtools}R Documentation

Computation and formatting of CIs for glm

Description

glmCI returns a list with coefficient,CIs, and coef [CIs].

Usage

glmCI(model, min = 0.01, max = 100, cisep = "<U+22EF>", ndigit = 2)

Arguments

model

Output from glm .

min

Smallest OR to report.

max

Largest OR to report.

cisep

Separator between CI values.

ndigit

rounding level.
