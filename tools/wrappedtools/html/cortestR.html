cortestR {wrappedtools}R Documentation

correlation matrix with p-values based on cor.test

Description

cortestR returns a data frame with coefficient,p-values, and significance symbols.

Usage

cortestR(
  cordata,
  method = "pearson",
  digits = 3,
  digits_p = 3,
  sign = "symbol"
)

Arguments

cordata

data frame or matrix with rawdata.

method

as in cor.test.

digits

rounding level for estimate.

digits_p

rounding level for p value.

sign

if 'symbol', add significance indicator.
